# **App Name**: مصنف النصوص

## Core Features:

- إدخال النص و الإرسال: واجهة المستخدم لإدخال النص و زر لإرسال الاستعلام إلى واجهة برمجة تطبيقات Hugging Face.
- تكامل Hugging Face AI: تستخدم هذه الأداة نقطة نهاية Hugging Face API لتحليل المدخلات النصية وإنشاء تنبؤات AI، مع تضمين المعالجة وإرسال الاستعلام.
- عرض التنبؤ: عرض نظيف وبسيط لنتائج التنبؤ التي تم إرجاعها من واجهة برمجة تطبيقات Hugging Face.
- مؤشر التحميل: عرض رسالة "جاري المعالجة..." أثناء انتظار الاستجابة من واجهة برمجة تطبيقات Hugging Face.
- معالجة الأخطاء: التعامل مع أخطاء API وعرض رسائل ذات صلة للمستخدم.
- تصميم RTL وواجهة عربية: واجهة المستخدم بالكامل مكتوبة باللغة العربية الطبيعية، مع تخطيط صارم من اليمين إلى اليسار (RTL).
- روابط اجتماعية وتكريم: قسم سفلي أنيق يتضمن تكريماً لـ Mahmoud Albahrani وأزرار منسقة بشكل أنيق لـ LinkedIn و GitHub مع روابط لـ https://www.linkedin.com/in/mahmoudbahrani/ و https://github.com/Mahmoud5150.

## Style Guidelines:

- Primary color: A sophisticated deep violet-blue (#513F8A) that anchors the interface with professionalism and technological depth, ensuring high contrast on lighter elements.
- Background color: A serene, ultra-light lavender-grey (#F7F6F8) providing a clean, expansive canvas that evokes modernity and a minimalist aesthetic.
- Accent color: A vibrant, clear sky blue (#61ADDE) to highlight interactive elements, calls-to-action, and status indicators, drawing the user's eye without being overwhelming.
- Social Link Styling (LinkedIn): The LinkedIn button should adhere to the platform's brand blue, integrated seamlessly within the overall design palette.
- Social Link Styling (GitHub): The GitHub button will feature a sleek, dark aesthetic to represent the platform's identity, providing a strong visual counterpoint in the footer.
- Body and headline font: 'Inter' (sans-serif), selected for its modern, objective, and neutral aesthetic. For accurate and high-quality Arabic text rendering throughout the strictly Arabic UI, a complementary, high-quality Arabic sans-serif font should be chosen to maintain this aesthetic.
- Utilize minimalist SVG icons for all functional elements, especially the social media links, to maintain a sharp, contemporary look that is easily scalable across devices.
- A strictly Right-to-Left (RTL) layout implemented throughout the application to properly support Arabic, with responsive spacing, soft shadows, and rounded corners applied uniformly for a sleek and elegant mobile-first design.
- Incorporate subtle hover and transition effects on buttons and interactive components, particularly the social media links, to provide delightful user feedback. A smooth, gentle animation for the loading indicator will enhance the perceived speed and user experience.
